Julienne on GitHub

The successor to the TAU app, Julienne is an application designed for inventory management and tracking. The project is ongoing but will eventually offer both a desktop application and web-app framework.

The desktop component is built with the ElectronJS library, and uses NodeJS to interact with the local OS. The UI is built with React and data storage is held in the Firebase cloud.

Features

  • Cloud Functions
  • Desktop Environment
  • CRUD operations
  • Account management
https://github.com/BSickler/julienne
TEMPO Automation Utility

TAU was created to simplify some of the daily tasks during my time at UCSB's Materials Research Lab. The application tracks inventory and maintenance records while automating notifications.

The code relies heavily on DOM manipulation and utilizes cloud functionality. User authentication, serverless functions and data storage are all contained in the Firebase ecosystem.

Features

  • DOM manipulation
  • User permissions
